# ロリポップ！マネージドクラウドへのデプロイガイド (FastAPI)

FastAPIバックエンドをマネージドクラウド上で常駐させ、どこからでも解析機能を使えるようにする手順です。

## ステップ 1: マネージドクラウドでのプロジェクト作成
1. **[ロリポップ！マネージドクラウド公式サイト](https://mc.lolipop.jp/)** にアクセスします。
   - ※注意：通常のロリポップ（レンタルサーバー）の管理画面とはURLが異なります。
2. 画面右上の **「ログイン」** ボタンをクリックします。
3. 登録したメールアドレスとパスワードでログインすると、そこが **「マネージドクラウドのダッシュボード（管理画面）」** です。
4. **「新しくプロジェクトを作成」** をクリックします。
5. 言語で **「Python」** を選択して作成を完了させます。
4. **【重要】** SSH公開鍵を登録します（右上のユーザーアイコン > SSH公開鍵）。これがないとGitプッシュができません。
5. 作成したプロジェクトの **「詳細ページ」** を開き、画面下部の **「SSH / SFTP」** セクションを探します。
6. その中にある **「リポジトリ」** という項目に表示されているGitのURL（`ssh://...`）をコピーします。

## ステップ 2: デプロイ用ファイルの確認
プロジェクトの `backend/` ディレクトリに必要なファイルが揃っていることを確認してください。

- `main.py`: バックエンド本体
- `requirements.txt`: 必要なライブラリリスト（作成済み）

## ステップ 3: 起動コマンドの設定
マネージドクラウドのダッシュボード画面で、**「起動コマンド」** を以下のように設定します。

```bash
uvicorn main:app --host 0.0.0.0 --port 8080
```

## ステップ 4: コードのプッシュ (デプロイ)
手元のPCのターミナルで、`backend/` フォルダの中身をマネージドクラウドのリポジトリにプッシュします。

```bash
# 1. バックエンドディレクトリへ移動
cd backend

# 2. Gitの初期化とコミット
git init
git add .
git commit -m "Initial deploy to Managed Cloud"

# 3. リモートリポジトリの追加（コピーしたURL）
git remote add mc [コピーしたGitリポジトリURL]

# 4. プッシュしてデプロイ
git push mc master
```

## ステップ 5: フロントエンドからの接続設定
1. デプロイ完了後、発行されたプロジェクトURL（例: `https://[英数字].mc.lolipop.jp`）をコピー。
2. [動画解析ツール](https://tajmahal.mond.jp/research/video_analyzer/)の **[設定]** で、このURLを入力してください。

---

> [!IMPORTANT]
> **OpenCV の注意点:**
> サーバー環境では GUI がないため、通常の `opencv-python` ではなく `opencv-python-headless` を使用する必要があります。
> すでに `backend/requirements.txt` にはその設定を反映済みです。
